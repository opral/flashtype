- Flashtype can now open any folder without crashing.
- Opening folders no longer writes `.lix` workspace data by default.
- Use Track Changes from the File menu or Dock menu to save change-tracking metadata into a folder.
- Large folders open faster and no longer show the folder size warning.
- Improved CSV viewing for wide spreadsheets.
- CSV links are now clickable.
- New Markdown files now start as a heading.
